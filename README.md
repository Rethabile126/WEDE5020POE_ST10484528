## Project Overview
A 5-page static website for Leona’s Cakery, a small bakery in Pretoria. Built using HTML5 and CSS only for Part 1 of the POE.

## Website Goals and Objectives
1. Establish an online presence for Leona’s Cakery
2. Reduce enquiry response time with a structured contact form
3. Improve customer experience with clear pricing, gallery, and map
4. Build brand credibility through consistent design
5. Meet WEDE5020 Part 1 outcomes for HTML5 and external CSS

## Key Features and Functionality
- 5 linked pages: Home, About, Services, Enquiry, Contact
- Semantic HTML5: `<header>`, `<nav>`, `<main>`, `<footer>`
- External stylesheet `CSS/style.css` for all styling
- Fixed sidebar navigation with active page highlighting
- Responsive images with alt text
- HTML enquiry form with validation
- Google Maps embed on Contact page
