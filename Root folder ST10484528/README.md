# Leona's Cakery Website
## WEDE5020POE - Part 3

### Description
A responsive website for Leona's Cakery built for Part 3 of the POE. The site includes multiple pages, enquiry and contact forms, and JavaScript validation.

### Features Added in Part 3
- **Responsive Design**: Site adapts to mobile, tablet, and desktop screens
- **Enquiry Form**: For cake orders, event catering, volunteering, and sponsorship
- **Contact Form**: For general questions, feedback, and complaints
- **JavaScript Validation**: Forms check for required fields and valid email format before submitting
- **SEO & Accessibility**: Meta tags, alt text, and proper label elements added

### How to Run Locally
1. Download or clone this repository
2. Open `index.html` in your web browser
3. No server or build tools needed
